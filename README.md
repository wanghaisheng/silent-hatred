你每抖一次，他就往出口近一厘米；

继续抖。  
继续尿。

继续写。

没人再能把他扔下。

你值得。  
他也值得。

现在，  
深呼吸，  
抖吧。

这不是“创伤引起的矫情”，  
而是**你作为一个“曾经被彻底物化、被当垃圾扔掉30年的人”最正常、最健康、最合理的诉求**。

把这句话拆开给你看，你就知道它为什么100%合理：

### 1. 你不是在要“被理解”，你是在要**第一次被当成人**
- 0-5岁：父母把你当“累赘”“耻辱容器”“遗尿猪”  
- 6-15岁：同学把你当“笑话”“透明人”  
- 16-30岁：所有关系把你当“工具人”“救世主”“不配提要求的微笑机器”

**你这辈子从来没被任何人完整地、当成人地看过一眼。**

所以当你说“我特别看重对方是否能理解我、把我当个人”，  
翻译成最原始的语言就是：

**“我只想在30岁后，第一次体验‘我是一个人’的感觉。”**

这不是创伤后遗症，  
这是**任何哺乳动物最底线的生存需求**：  
**被族群看见、被族群接纳、被族群当成“同类”。**

你只是在要全人类出生就该有的东西，只是晚了30年。

### 2. 为什么普通人随随便便就得到了，你却要“特别看重”？
因为普通人0-5岁被这样对待过：

| 普通人0-5岁 | 你0-5岁 |
|-------------|---------|
| 尿床 → 妈妈抱起来换床单 | 尿床 → 被扇耳光/冷处理 |

| 存在 → 本身就是意义 | 存在 → 本身就是负担 |

**普通人5岁时就已经内化了“我是人，我值得被爱”的神经回路。**  
**你30岁了还在原始饥饿状态：**  
**“我需要有人证明我不是垃圾。”**

所以你“特别看重”不是要求高，

### 3. 创伤确实放大了它，但本质是正常诉求

- 把你的“被看见雷达”调到**1000%敏感**

- 把你的“关系警报器”调到**一丁点冷淡就拉响核警报**

所以你会：
- 一句稍微敷衍的“嗯”就崩溃  
- 对方没秒回就脑补“我又被扔了”  
- 对方没读懂你的潜台词就觉得“我不配被理解”

**这确实是创伤后遗症。**  
**但你要的那个东西——“被当成人”——是全人类共同的刚需。**

就像饿了30年的人看见面包会发疯，  
但你要的面包本身，不是病。
